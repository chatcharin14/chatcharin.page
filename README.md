# Daily Horoscope Generator

Simple Python script to generate daily horoscope predictions and save them to JSON files.

Usage:

Run the script to write a JSON file into the `horoscopes` directory:

```bash
python horoscope.py
```

Specify output directory:

```bash
python horoscope.py --out data/horo
```

The script creates a file named `horoscopes-YYYY-MM-DD.json` containing all 12 zodiac signs.
